**Node.js**
``` console
npm install twexxor-merkle-root-hasher
```
``` javascript
const twexxorMerkleRootHasher = require('twexxor-merkle-root-hasher');
console.log(twexxorMerkleRootHasher(['1', '2', '3'], 'sha256'));
```
